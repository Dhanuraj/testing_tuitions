🐙 GitHub Setup Guide

Connect your Tuition Quiz to GitHub — questions update on all devices instantly

STEP 1

Create a free GitHub account

Go to github.com and sign up for free. Takes 2 minutes.

📌 If you already have a GitHub account, skip to Step 2.
STEP 2

Create a new repository

After logging in:

1. Click the + button (top right) → New repository

2. Name it: tuition-quiz-questions

3. Set it to Public ✅ (important — quiz needs to read it)

4. Check "Add a README file"

5. Click Create repository

STEP 3

Upload questions.json

1. In your new repository, click Add file → Upload files

2. Drag and drop the questions.json file (download it below)

3. Scroll down, click Commit changes

⚠️ The file must be named exactly: questions.json
STEP 4

Get your raw file URL

1. Click on questions.json in your repo

2. Click the Raw button

3. Copy the URL from your browser — it will look like:

https://raw.githubusercontent.com/YOUR-USERNAME/tuition-quiz-questions/main/questions.json
